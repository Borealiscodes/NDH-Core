# 📄 **Adventure Generator Quickstart**  
### Continuity Adventure System — Forest of Echoes Edition  
A fast, gentle introduction to generating adventures from story fragments.

---

## 🌲 1. What This Quickstart Is  
This Quickstart teaches contributors how to:

- provide story fragments  
- run the Adventure Generator  
- understand the output  
- create a full adventure in seconds  

It is intentionally short, calm, and ND‑paced.

---

## 🧩 2. The Core Idea  
The Adventure Generator takes **small fragments** and produces a **full multi‑mode adventure**.

It builds:

- a **World Node**  
- a **Character Node**  
- a **Story Node**  
- a **reversible adventure loop**  
- a **four‑chapter progression**  
- optional symbolic systems  

Everything is continuity‑safe.

---

## 🌿 3. Minimal Input  
You can provide **one fragment**, or many.

### Example Minimal Input
```
"a soft hum behind a tree"
```

### Example Mixed Input
```
"a wanderer who hesitates before stepping forward"
"a pool that reflects sound instead of light"
"slow, eerie mood"
```

Fragments may be:

- world  
- character  
- story  
- mood  
- symbolic  

Any combination works.

---

## 🔮 4. What the Generator Produces  
From your fragments, the generator creates:

### World Node  
A structured environment.

### Character Node  
A reversible, ND‑paced character.

### Story Node  
A soft, reversible narrative moment.

### Adventure Loop  
Always:

```
World → Character → Story
```

### Chapters  
Four reversible chapters.

### Optional Systems  
Generated only if relevant:

- Echo Tokens  
- Resonance Map  
- Stillness Sanctuaries  

---

## 🌙 5. Quickstart Workflow  
This is the **fastest way** to generate an adventure.

### Step 1 — Provide Fragments  
Write any fragments you want.

Example:
```
"a soft hum behind a tree"
"a wanderer who hears future echoes"
"a gentle, looping mood"
```

### Step 2 — Run the Generator  
Use the generator command:

**Generate Random Adventure**

### Step 3 — Receive the Adventure  
You will get:

- World Node  
- Character Node  
- Story Node  
- 4 chapters  
- optional systems  

All reversible.  
All ND‑paced.  
All continuity‑safe.

---

## 🌌 7. Best Practices  
To keep adventures gentle and continuity‑safe:

- use symbolic hazards only  
- keep pacing slow and breathable  
- ensure scenes are reversible  
- avoid forced outcomes  
- maintain ND‑safe tone  
- keep characters non‑coercive  

These principles are built into the generator.
