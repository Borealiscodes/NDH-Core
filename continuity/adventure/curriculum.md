# 📄 **Adventure System Curriculum**  
### Continuity Adventure System — Structured Learning Path  
A complete, ND‑paced curriculum for learning, understanding, and contributing to the Continuity Adventure System.  
This curriculum is designed to be:

- gentle  
- reversible  
- symbolic  
- non‑coercive  
- paced for ND accessibility  

It guides contributors from foundational concepts to advanced architectural work.

---

# 🌲 **1. Curriculum Purpose**

The curriculum exists to:

- teach the Adventure System in a structured, safe way  
- help contributors understand each layer without overwhelm  
- provide reversible learning modules  
- support ND‑paced exploration  
- unify conceptual, architectural, and practical knowledge  

It is the **official learning path** for the Adventure System.

# 🌿 **2. Curriculum Structure (Top‑Level)**

The curriculum is divided into **six learning tiers**:

1. **Orientation Tier** — What the system *is*  
2. **Node Tier** — World, Character, Story  
3. **Engine Tier** — Resonance, Stability, Narrative  
4. **Loop Tier** — Continuity architecture  
5. **Cosmology Tier** — Symbolic alignment  
6. **Contributor Tier** — Practical creation and collaboration  

Each tier is reversible and can be revisited at any time.
