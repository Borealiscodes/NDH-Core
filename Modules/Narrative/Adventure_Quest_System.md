# 📄 **Adventure / Quest System — Expanded Universe Engine**  
*(High‑Level Narrative Module)*

## 1. Purpose  
The Adventure / Quest System transforms the Story Node Engine and Faction Arc Generator into **playable narrative sequences**.  
It defines:

- how quests begin  
- how they branch  
- how they escalate  
- how they resolve  
- how consequences propagate  
- how cosmic, faction, economic, and identity pressures shape the journey  

This is the **player‑facing narrative layer** of the engine.

---

# 2. Core abstractions  

### **2.1 Quest Frame**  
A Quest Frame is the container for a full adventure arc. It includes:

- **Premise** — what starts the quest  
- **Objective** — what must be achieved  
- **Constraints** — faction, economic, identity, cosmic limits  
- **Risks** — combat, cosmic, political, cultural  
- **Rewards** — narrative, factional, economic, identity‑based  

### **2.2 Quest Path**  
A Quest Path is a sequence of Story Nodes connected by validated Node Links.

Paths can be:

- linear  
- branching  
- cyclical  
- escalating  
- collapsing  

### **2.3 Quest Trigger**  
Triggers initiate quests:

- faction tension spikes  
- cosmic anomalies  
- economic scarcity  
- identity conflicts  
- cultural rituals  
- combat escalation  

---

# 3. Quest schema  

Each quest includes:

- **Quest ID**  
- **Quest Frame**  
- **Trigger Node**  
- **Primary Path**  
- **Alternate Paths**  
- **Risk Profile**  
- **Reward Profile**  
- **Continuity Metadata**  
- **Generated Story Nodes**  
- **Generated Node Links**  

All quests are validated by the **Narrative Continuity Engine**.

---

# 4. Quest Generation Axes  
Quests evolve across **six axes**, each a Guided Link for deeper exploration:

- **Faction Pressure**  
- **Economic Scarcity**  
- **Identity Stakes**  
- **Cultural Imperatives**  
- **Cosmic Risk**  
- **Combat Challenge**  

These axes determine quest tone, difficulty, and branching structure.
