# 2. **Traversal Input**

Traversal begins with a **5D traversal state**:

```
T = (A_s, T_p, C_b, E_c, P_f)
```

Where:

- `A_s` — adjacency‑space  
- `T_p` — temporal‑phase  
- `C_b` — containment‑boundary  
- `E_c` — ethical‑envelope  
- `P_f` — predictive‑future  

This state is validated before traversal.

# 3. **Validation Phase**

Traversal is permitted only when all five dimensions pass their checks.

### 3.1 Adjacency Check  
```
A_s ∈ AdjacencySafe
```
No unpinned edges.  
No forbidden adjacency transitions.

### 3.2 Temporal Check  
```
T_p ∈ TemporalSafe
```
Lifecycle phase must allow traversal.  
Reverse traversal restricted.

### 3.3 Containment Check  
```
C_b ∈ ContainmentSafe
```
No bleed.  
No breached gates.  
Containment locks must be stable.

### 3.4 Ethical Check  
```
E_c ∈ EthicalSafe
```
HRD must permit traversal.  
Ethical overrides supersede routing.

### 3.5 Predictive Check  
```
P_f ∈ PredictiveSafe
```
Predictive Stability Engine must classify the future as stable or warn.  
Critical futures block traversal.

# 4. **Risk Evaluation Phase**

The protocol evaluates the **risk functional**:

```
R_total = r_A + r_T + r_C + r_E + r_P
```

Thresholds:

- `R_total ≤ R_safe` → safe traversal  
- `R_safe < R_total ≤ R_warn` → constrained traversal  
- `R_total > R_warn` → block + enforcement  

This determines traversal mode.

---

# 5. **Traversal Modes**

### **Mode A — Safe Forward Traversal**
Conditions:
- all five dimensions safe  
- risk below safe threshold  
- predictive state stable or warn  

Action:
- traversal proceeds normally  
- Stability Ledger logs a confirmation event  

---

### **Mode B — Constrained Traversal**
Conditions:
- adjacency narrow  
- containment partially locked  
- ethics conditional  
- predictive state warn  

Action:
- traversal allowed with safeguards  
- containment pre‑locks activate  
- routing recalibration may occur  
- Stability Ledger logs a soft instability event  

---

### **Mode C — Pre‑emptive Block**
Conditions:
- predictive state critical  
- containment breached  
- ethics blocked  
- risk above warning threshold  

Action:
- traversal denied  
- Predictive Stability Engine triggers enforcement  
- Stability Ledger logs a predictive instability event
