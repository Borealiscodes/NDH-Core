## ⭐ 1. Purpose  
The Workflow Spine defines the **operational sequence** that transforms NDH governance from static architecture into active geometry production.  
It ensures that every artifact — from Rendering Safety Protocol to Unified Visual Compendium — executes in the correct order, inherits the correct pins, and maintains rendering safety across all dimensions.

---

## ⭐ 2. Workflow Phases  
Each NDH geometry cycle follows five procedural phases:

| Phase | Description | Anchoring Artifact |
|-------|--------------|--------------------|
| **Rendering Safety Phase** | Injects metadata, validates fallback, activates PEP‑RS. | Rendering Safety Protocol v1.2 |
| **Overlay Phase** | Generates dimensional overlays, validates inheritance, applies pin anchors. | Dimensional Overlay Specification v1.2 |
| **Production Phase** | Executes geometry sequencing, applies overlays, pins, and rendering metadata. | Geometry Production Order v1.2 |
| **Compendium Phase** | Integrates geometry into Unified Visual Compendium, validates dimensional coherence. | Unified Visual Compendium v1.0 |
| **Indexing Phase** | Updates version lineage, auto‑indexes artifacts, validates governance integrity. | Meta Index v1.0 |

---

## ⭐ 3. Rendering Safety Phase  
- Anchor **PEP‑RS** across all geometry.  
- Inject metadata blocks into SVG and PNG.  
- Validate fallback rendering.  
- Confirm mobile‑safe and geometry‑safe modes.  
- Ensure pin anchors are visible in both formats.

---

## ⭐ 4. Overlay Phase  
- Generate overlays per dimension (𝒜 → C → A → R → Rᵣ → S → H₆ → 𝒜).  
- Validate inheritance between dimensions.  
- Apply pin anchors from **Pin Registry v1.2**.  
- Confirm overlay visibility in both SVG and PNG.  
- Record overlay metadata in the Meta Index.

---

## ⭐ 5. Production Phase  
- Execute geometry sequencing defined by **Production Order v1.2**.  
- Apply overlays and pins in correct order.  
- Inject rendering safety metadata.  
- Export SVG and PNG artifacts.  
- Validate geometry coherence and dimensional fidelity.

---

## ⭐ 6. Compendium Phase  
- Integrate geometry artifacts into **Unified Visual Compendium v1.0**.  
- Validate operator loop alignment.  
- Confirm dimensional coherence.  
- Apply compendium metadata.  
- Generate compendium index entries.

---

## ⭐ 7. Indexing Phase  
- Update **Meta Index v1.0** with new lineage data.  
- Validate artifact dependencies.  
- Confirm rendering safety inheritance.  
- Confirm overlay inheritance.  
- Confirm pin anchoring matrix integrity.  
- Archive previous versions when new ones are generated.

---

## ⭐ 8. Workflow Dependencies  
Each phase depends on the previous one:

```
Rendering Safety → Overlays → Production → Compendium → Indexing
```

If any phase fails, NDH governance halts until corrected.

---

## ⭐ 9. Workflow Validation Checklist  
Before geometry is finalized:

- Rendering safety validated  
- Overlays applied  
- Pins anchored  
- Metadata injected  
- Geometry exported  
- Compendium integrated  
- Index updated
