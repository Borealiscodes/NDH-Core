## Purpose
The Math Check Protocol™ defines the verification steps required before any mathematical structure becomes canonical within UMA. It ensures that new math is coherent, constitutional, non‑drifting, and aligned with established provenance.

This protocol prevents mathematical distortion, false symmetry, lineage contamination, and runtime instability.
